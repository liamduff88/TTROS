# /business-efficiency-assessment
> Revisit: if assessment pricing, report structure, or matrix method changes. · Last touched: 2026-07-07

## Purpose
Produce the executive six-part assessment report that maps a client's workflow bottlenecks and routes them into the right builds or support.

## Inputs
- Client entity page + discovery interview notes/transcript (Liam runs the interviews personally at this stage — do not automate intake).

## Steps
1. **Inventory** — list candidate workflows from the notes: lead response, missed calls, CRM handoff, follow-up, intake/quoting, reporting, knowledge sprawl, repeated admin.
2. **Select 3–7 bottlenecks** — high-frequency × high-friction wins. Each tagged to an ROI lever.
3. **Matrix** — effort-vs-impact by default; impact-vs-risk where workflows touch sensitive data. Tag each item manual / hybrid / automated as the target state.
4. **Classify fixes** — per bottleneck: quick win, workflow redesign, tool/integration, controlled agent or client memory system, stays-human, or hybrid with approval points.
5. **Draft report** — six parts: executive summary · opportunity matrix · recommended tools/systems with cost ranges · quick-win plan · implementation/upsell opportunities · quantified value estimate with assumptions labelled.
6. **Route** — end with the walkthrough question ("implement internally / support / hand off") and map recommendations to offers: AI Operations Support, Speed-to-Lead, Voice Agent, Client Memory, Lead Gen Agent, training, custom, agentic management.
7. **Close** — artifact + receipt (token breakdown) + entity page update. Target: report within 48h plus 30-min walkthrough.

## Never
- Fewer than 3 or more than 7 headline bottlenecks.
- Unlabelled assumptions in the value estimate; guaranteed savings/revenue claims.
- Recommending full autonomy where a human approval gate belongs.
- Generic AI-agency framing — voice stays calm, premium, systems-led.

## Done when
Six-part report artifact exists, every bottleneck tagged (lever + matrix position + target state), next-step routing included, receipt written.
