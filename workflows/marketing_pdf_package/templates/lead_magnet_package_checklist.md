# Lead Magnet Package Checklist

## Intake

- [ ] Approved Marketing draft is present.
- [ ] Approval evidence is recorded.
- [ ] Audience and offer context are clear.
- [ ] CTA is approved or flagged for Liam review.

## Package

- [ ] Title is specific and useful.
- [ ] Subtitle clarifies the outcome.
- [ ] PDF source markdown is complete.
- [ ] CTA appears in the right place.
- [ ] Examples and proof points are checked.
- [ ] Claims are practical and defensible.

## Render Handoff

- [ ] Markdown source is ready for `workflows/pdf_branding`.
- [ ] Desired output filename is documented.
- [ ] Page break notes are documented.
- [ ] Render receipt path is documented.

## Final Review

- [ ] Final PDF or HTML fallback exists.
- [ ] Formatting has been checked.
- [ ] CTA has been checked.
- [ ] Receipt is complete.
- [ ] Liam completed human review before publishing.
